  ---Instruçoes de uso---
  Artefato: Cadastro e/ou Perfil de Usuário (sistema para cadastrar perfiz, edita-los ou exclui-los).
  
  1. Clonar o diretório 'Lucas'.
  2. Abrir a página 'index.html'.
  3. Para adicionar um novo perfil preencha o formulario de cadastro envie. Informações necessárias: nome do usuario, senha e email.
  4. Para carregar todos os dados de perfiz ja cadastrados é só clicar no botão de 'Carregar dados'.
  5. Para alterar o email, que é o unico dado que é possivel alterar, por enquanto, logue na conta e clique no botão de 'Deseja alterar seu email?'.
  6. Para excluir o perfil é só clicar em 'Excluir conta'.
  7. Para sair da conta é só clicar no botão 'Sair'.
  
  Todas as informações como a lista de jogos cadastrados serão salvas no local storage do navegador.
